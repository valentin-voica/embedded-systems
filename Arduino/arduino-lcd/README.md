# Connecting an LCD to an Arduino Board

How to connect an LCD to Arduino

## LCD 1602A with I2C LCD Backpack

Connect the I2C Module to the Arduino:

SCL of module to A5 of Arduino
SDA of module to A4 of Arduino
VCC of module to +5V of Arduino
GND of module to GND of Arduino

Adjust the brightness using the small, blue coloured potentiometer.

See the test code.
