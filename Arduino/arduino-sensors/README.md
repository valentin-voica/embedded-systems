# Arduino Sensors

How to connect sensors (temperature, humidity, etc) to Arduino

## DHT11, DHT22, AM2302

Follow the instructions on [Adafruit website](https://learn.adafruit.com/dht/overview).

If using a DIY Arduino, there is no "Serial Monitor". If using the Pololu AVR programmer, connect a jumper-wire from the "Tx" pin on the Arduino board to the "Rx" pin on the programmer.
